Lumen Light static highlight demo

Select any text in this document to create a gold highlight. Click an existing highlight to remove it. Highlights persist in this browser for this page.

A partner process can also add a purple highlight through the public browser API. This keeps visual collaboration separate from the raw text while still allowing the page to export durable highlight records.

The important architectural point is that the document remains plain HTML. Lumen Light adds reversible presentation state and a small structured export surface without taking ownership of long-term memory.
